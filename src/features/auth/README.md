## Фича авторизации

Описание:
содержит кнопку авторизации и логику выхода пользователя.

#### Public API

- Components

`AuthButton` - кнопка авторизации пользователя

- Hooks

`useLogout` - хук для выхода из аккаунта

#### Пример

```ts
import { AuthButton, useLogout } from "@/features/auth";
```
