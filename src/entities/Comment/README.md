## Сущность комментария

Описание:
содержит тип комментария и UI для отображения комментариев.

#### Public API

- Types

`Comment` - тип комментария

- Components

`CommentCard` - карточка комментария

`CommentCardSkeleton` - скелетон карточки комментария

#### Пример

```ts
import { CommentCard, type Comment } from "@/entities/Comment";
```
