## Сущность уведомления

Описание:
содержит API-запросы и UI для отображения уведомлений.

#### Public API

- API

`useNotificationsQuery` - хук для получения списка уведомлений

- Components

`NotificationItem` - элемент списка уведомлений

#### Пример

```ts
import { NotificationItem, useNotificationsQuery } from "@/entities/Notification";
```
