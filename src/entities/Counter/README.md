## Сущность счетчика

Описание:
содержит пример простой redux-сущности счетчика.

#### Public API

- Types

`CounterSchema` - схема состояния счетчика

- Model

`counterReducer` - редьюсер счетчика

- Components

`Counter` - UI счетчика

#### Пример

```ts
import { Counter, counterReducer, type CounterSchema } from "@/entities/Counter";
```
