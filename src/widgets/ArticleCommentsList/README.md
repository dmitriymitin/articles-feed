## Виджет со списком комментариев к статье

Описание:
содержит список комментариев к статье и API для обновления комментариев.

#### Public API

- API

`refetchArticleCommentsListQuery` - повторный запрос списка комментариев

- Components

`ArticleCommentsList` - список комментариев к статье

#### Пример

```ts
import { ArticleCommentsList } from "@/widgets/ArticleCommentsList";
```
