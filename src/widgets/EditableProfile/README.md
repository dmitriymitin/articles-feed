## Виджет редактирования профиля

Описание:
содержит форму редактирования профиля пользователя.

#### Public API

- Types

`ProfileSchema` - схема состояния формы профиля

- Components

`EditableProfileCard` - карточка редактирования профиля

#### Пример

```ts
import { EditableProfileCard, type ProfileSchema } from "@/widgets/EditableProfile";
```
