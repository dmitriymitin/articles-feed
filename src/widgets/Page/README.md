## Виджет страницы

Описание:
содержит обертку страницы и состояние UI для страниц.

#### Public API

- Types

`UISchema` - схема UI-состояния страницы

- Model

`uiReducer` - редьюсер UI-состояния страницы

- Components

`Page` - обертка страницы

#### Пример

```ts
import { Page, uiReducer, type UISchema } from "@/widgets/Page";
```
